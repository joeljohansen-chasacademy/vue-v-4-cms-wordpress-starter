# Detta är ett starterrepo för att komma igång med Wordpress och Docker.

För att starta, klona repot och kör:

```bash
docker-compose up -d
```

Detta kommer att installera och starta en Wordpress-instans tillgänglig på `http://localhost:8080` och en MySQL-databas. (Se docker-compose.yml för konfigurationsdetaljer.)

I wordpress utvecklar man "teman". Teman ligger i mappen `wp-content/themes/`.
